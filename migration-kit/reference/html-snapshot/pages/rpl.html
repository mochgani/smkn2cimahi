Rekayasa Perangkat Lunak

Kompetensi RPL membekali siswa keterampilan pengembangan perangkat lunak dengan tiga konsentrasi utama: web, desktop, dan mobile.

RPL
[01] Tentang

Tentang program keahlian

Kompetensi Rekayasa Perangkat Lunak (RPL) adalah salah satu kompetensi keahlian dalam bidang Teknologi Komputer dan Informatika. Siswa dibekali keterampilan untuk mengembangkan tiga jenis perangkat lunak utama: aplikasi berbasis web, desktop, dan mobile (Android Apps).

RPL SMKN 2 Cimahi merupakan member dari Oracle Academy, memungkinkan siswa memperoleh sertifikasi internasional di bidang Java Fundamental, Java Foundation, dan Database Foundation.
